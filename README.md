# Friendshyp

This project is the frontend for the assessment.

## Project setup

1. Run `npm install` or `yarn`
2. Clone and setup [backend](https://github.com/Samueloyeks/friendshyp-backend)
3. Run `npm run start`
4. View app in `http://localhost:4200`
